### Boxoffice: HasGeek's ticketing system
[![Build Status](https://travis-ci.org/hasgeek/boxoffice.svg?branch=v0.2)](https://travis-ci.org/hasgeek/boxoffice)

## Asset builds

To run webpack in development mode:

```
    cd boxoffice/static/
    yarn dev
```

To build for production:

```
    cd boxoffice/static/
    rm build/*.* && yarn build
```
